Rescue Oscar 2 (Tatiana's 22nd B-DAY)

Instructions to download & play

Meet the Characters

1. Download the Game

Download Game ZIP

2. Requirements

  • Java Development Kit (JDK)
    • Go to https://adoptopenjdk.net/
    • Download & install “OpenJDK 8 (LTS)” for your OS.
    • Follow the installer's prompts (accept defaults).
    • Verify installation:
      — Windows: Open a Command Prompt and run:
        java -version and javac -version
      — macOS: Open Terminal and run:
        java -version and javac -version
      You should see version output rather than “command not found." Restart your PC if it's not there.
  • Greenfoot 3.x
    • Visit greenfoot.org/download
    • Download the package for your OS.
    • Run the downloaded installer.

3. Installation & Running

  1. Unzip the Game
    • Locate the downloaded .zip.
    • Right-click → “Extract All…” (Windows) or double-click (macOS/Linux).
  2. Launch Greenfoot
    • Open the Greenfoot app you just installed.
  3. Import the Project
    • In Greenfoot choose Scenario → Open…
    • Navigate to the folder you extracted.
    • Select it and click “Open”.
  4. Run
    • Click the ▶︎ Run button at bottom.
    • Once the home screen animates, you're good to go!
